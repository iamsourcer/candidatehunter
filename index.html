What it does

Everything a recruiter
actually needs

Built for technical recruiters who review dozens of profiles a day and need fast, reliable signal — not more noise.

⚡

Auto-analyze on open

Navigate to any LinkedIn or Ashby profile and the extension analyzes it automatically. Badge shows green/red + match % without clicking anything.

🧠

AI-powered evaluation

Uses Claude, GPT-4, DeepSeek or Gemini to score each candidate against your role requirements. Generates structured ADVANCE/ARCHIVE verdicts.

📋

Phone screen scripts

For ADVANCE candidates, AI writes a custom interview script targeting the specific gaps it found in their profile. No generic questions.

🏢

Multi-ATS support

Works on LinkedIn profiles and Ashby ATS candidate pages. Select resume text in Ashby for richer analysis.

🗂

Projects & lists

Organize candidates into named projects. View full AI analysis, download JSON exports, or push results back to Ashby as notes.

🎭

Role configurations

Save multiple evaluation prompts for different open roles. Switch active role in seconds — each with its own company context and criteria.

Installation instructions

The extension isn't in any store yet — install it manually in under 2 minutes.

  1. Download candidatehunter-chrome.zip and unzip it to a folder.
  2. Open Chrome and go to chrome://extensions
  3. Enable Developer mode (toggle in the top-right corner).
  4. Click "Load unpacked" and select the unzipped folder.
  5. The 🎯 CandidateHunter icon appears in your toolbar. Click it → Settings → add your AI API key.
  1. Download candidatehunter-firefox.zip and unzip it to a folder.
  2. Open Firefox and go to about:debugging
  3. Click "This Firefox" in the left sidebar.
  4. Click "Load Temporary Add-on…" and select the manifest.json file inside the unzipped folder.
  5. The 🎯 CandidateHunter icon appears in your toolbar. Click it → Settings → add your AI API key.
  6. Note: Temporary add-ons are removed when Firefox restarts. For a permanent install, the extension needs to be signed by Mozilla.
